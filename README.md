## Deployment

To deploy this project run

1) Clone this repository 

2) If you have not installed Vue.js run 
```bash
  npm install -g @vue/cli
```
3) To run the code 
```bash
  npm run serve
```
4) In case the code is giving an error run 
```bash
  npm i
```
